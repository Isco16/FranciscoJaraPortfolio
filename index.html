PROJECTS

    I have created several projects for different platforms, such as: mobile, VR, desktop, web, totems and arcade devices. In addition, I have experience using engines like Unity, Spark AR, 3D Vista Studio, and knowledge in libraries such as Open CV and Vuforia.

2021

MUI Las Edades de la Tierra

Project made with Unity and with a Kinect device integration. The software consists of a runner videogame made for “Museo interactivo de las Condes” Museum for scientific dissemination purposes. The project’s key feature is that players can use their body movements as the game input. There is also a keyboard version of the game. I was in charge of programming and design of the game mechanics.

2022

VR Moto Lenovo

Virtual reality moto racing game. Made with Unity with Oculus Quest 2 integration and a simulation motorcycle. Consist of an advergame which advertise Lenovo’s new gamer notebooks along with Xbox Live service. The objective of this project was create an immersive experience where players could participate on a virtual racing using a real Ducati motorcycle as input for the game. My contribution on this game was programming the game mechanics.

2022

Embajadores de la seguridad vial

Educational Project worked in collaboration with Conaset. This app is an interactive software in which players, or students, can learn about road safety regulations through a series of mini-games. Made with Unity engine and targeted to mobile and web platforms. I was responsible for programming every mini-game mechanics as well as giving support to the back-end connection part.

2020

Molinai

Augmented reality application developed for “Molinai, el encuentro con la naturaleza” book. The mobile app works together with this book in which users can get extended content when using the device camera at book’s illustrations. It was built with Unity engine and Vuforia technology. I was in charge of the software programming and the augmented reality content integration.
